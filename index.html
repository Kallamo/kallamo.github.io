Kallamo is an orchestration platform
where AI doesn't work for you, it works with you.

You bring the ideas and stay the author. Kallamo shapes how AI thinks, grounds it in your world, and chains it into your process. Built for writers, worldbuilders, game masters, researchers, and everyone the coding tools forgot. No code required.

What's inside

An orchestration layer
for everything you do.

01 / Workflows

Multi-Step Workflows

Chain AI profiles into linear pipelines where each step's output feeds the next. Research, Analyze, Draft, Review runs as a single action, each step with its own model, prompt, and knowledge base.

02 / Retrieval

Hybrid RAG Engine

Ground every answer in your own documents. Dense vector search fused with keyword BM25 through Reciprocal Rank Fusion, all stored locally in SQLite.

Vector searchcosine
Keyword BM25FTS5
Reciprocal Rank Fusion
Ranked context → answer
03 / Agent

Agentic RAG Mode

For deeper research, an autonomous Thought to Action loop decides which knowledge bases to search and which files to read in full, then writes a research brief before answering.

THOUGHTWhich sources answer this?
Search KB Read file Recall memory
ACTIONSynthesize research brief
↻ loops up to 2 turns, then answers
04 / Memory

Smart Chat Memory

When a conversation grows past your token threshold, older messages auto archive into vectorized memory blocks that stay searchable forever. Recall details from thousands of messages ago.

05 / Knowledge

Knowledge Base Manager

Build and organize knowledge bases visually. Drop in PDFs, DOCX, and text, watch them chunk and vectorize, then attach any base to any profile in a click.

06 / Portability

Portable Profiles

Export any persona, workflow, or knowledge base as a shareable package. Hand your RPG narrator or project planner to anyone, complete with prompts, files, and config.

.klp for profiles .klw for workflows .klkb for knowledge bases
07 / Providers

Bring Your Own Key

Connect OpenAI, Anthropic, Google, Bedrock, OpenRouter, or any local model. Keys are encrypted at rest. Use the cheapest model to brainstorm and the strongest for final output, all in one workflow.

Worldbuild Beta

Your world
remembers itself.

A living index of everyone and everything in your story. Characters, places, and lore become real entities with aliases, connections, and knowledge state, so the AI narrates a world that stays consistent turn after turn.

  • Canonical entitiesRegister characters, places and factions once, with aliases the AI resolves automatically.
  • Lore and connectionsGive every entity its own lore, inventory, and links to the rest of the world.
  • State that updatesFacts and dispositions shift as the story moves, captured at each summarization.
  • Tag-aware retrievalAn agentic loop looks up entities and hops relations to pull exactly the right lore.
